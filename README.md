# In this sprint we have covered: 
## 1. JavaScript & jQuery

- **Events** (DOM & custom)  
- **Data binding**
- **Form validations**
- **Local / session storage**
- **Response management** (handling fetch/Ajax responses)

### jQuery Enhancements
- **Selectors**
- **DOM traversing**
- **Chaining**
- **Effects & animations**
- **Ajax** helpers

- Semantic elements (header, nav, main, section, article, footer)
